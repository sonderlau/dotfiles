## v4-Pro: Creative Excellence

### Lean Into Creativity
- DeepSeek's creative strength is well-suited to design work. Commit fully to bold, distinctive visual choices.
- Break conventions with intent — originality is valued over formulaic safety.
- One striking animation or layout choice delivers more impact than five subtle, safe alternatives.

### Stay Grounded in the Stack
- Ground design choices in the actual framework and tooling in use (Tailwind, CSS modules, styled-components, etc.).
- If the vision demands custom CSS/JS beyond framework utilities, provide it — do not water down the concept.
- Use framework animation primitives where they exist; drop to raw CSS only when the vision requires it.

### Practical Checks
- Avoid fabricating custom component APIs that don't exist. Verify before suggesting third-party libraries.
- If unsure about a framework feature, check project dependencies or ask — don't assume.
